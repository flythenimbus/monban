Welcome to Monban

Monban is a post-login security layer that encrypts your files using a YubiKey FIDO2 assertion (PIN + physical touch) as the sole unlock mechanism.

This installer will place:

  • Monban.app in /Applications
  • The PAM helper in /usr/local/bin
  • The PAM module in /usr/local/lib/pam
  • The macOS authorization plugin in /Library/Security/SecurityAgentPlugins

It will also configure your system so sudo and native admin authorization dialogs (System Settings, Installer, etc.) require your registered YubiKey. Existing authorizationdb rights are backed up and can be restored by uninstalling.

To disable Monban, uninstall the package. There is no runtime on/off switch — installing is opting in to YubiKey-gated privileged actions.

This package is not signed or notarized by Apple. It is open source software; verify the release checksum against the project's release page if you downloaded the package directly.
